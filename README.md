# rmstyls
python program for removing unused jsx style codes from your project

Example:

unusedStyle: {
  flex: 1,
},
container: {
  flex: 1,
  paddingHorizontal: 10,
  backgroundColor: colors.page_bg_color,
},
emptyText: {
  textAlign: 'center',
  marginTop: 20,
},
unusedStyle2: {
  flex: 1,
},
lottieStyle: {
  alignSelf: 'center',
  width: 70,
  height: 70,
},
unusedStyle3: {

},
